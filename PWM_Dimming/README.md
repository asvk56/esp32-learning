# 04 PWM Dimming (ESP32 v3.0+)

- **Concept:** Pulse Width Modulation and Duty Cycle Control.
- **Hardware:** External LED on GPIO 22 with 220-ohm resistor.
- **Learning:** Utilizing the new `ledcAttach()` and `ledcWrite()` API in ESP32 Core v3.0+ to smoothly control LED brightness autonomously without CPU overhead.
