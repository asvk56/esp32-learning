# 05 Analog Read (ADC)

- **Concept:** 12-bit Analog to Digital Conversion.
- **Hardware:** Potentiometer wiper connected to safe ADC1 pin (GPIO 34).
- **Learning:** Reading continuous voltage ranges from 0V to 3.3V, mapped dynamically from 0 to 4095 steps. Implemented precision floating-point arithmetic to prevent integer division truncation.
